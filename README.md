# taskmr
task manager for CLI

# Install

```
cargo install taskmr
```

# Usage

```
$ taskmr help
taskmr
Task ManageR

USAGE:
    taskmr <SUBCOMMAND>

OPTIONS:
    -h, --help    Print help information

SUBCOMMANDS:
    add      Add a task
    close    Close tasks
    edit     Edit the task
    help     Print this message or the help of the given subcommand(s)
    list     List tasks
```
